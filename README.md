## Project Idea
This project helps create a greener campus by tracking electricity and water usage in campus buildings.
Students or staff can log daily usage, identify waste, and promote energy-saving habits.

## How It Helps the Environment
- Encourages awareness of energy consumption
- Helps reduce unnecessary electricity and water usage
- Supports sustainability goals on campus

## Technologies Used
- HTML (basic interface)
- Python (data processing)
- CSV / Google Sheets (data storage)

## How to Run or Use
1. Clone the repository:
   git clone https://github.com/your-username/greener-campus-energy-tracker.git
2. Open `index.html` in a browser  
   OR  
   Run `tracker.py` using Python:
   python tracker.py
